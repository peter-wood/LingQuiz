Ling Quiz

This project aims to develop a custom quiz application for an introduction to Linguistics course we teach at the university of Saskatchewan. Right now it only provides a multiple choice interface, but the aim is to add tasks in which students are asked to do phonetic transcriptions, morpheme analysis, sentence analysis, etc.

The project uses:

  • Node JS
  • Mongo DB

Please feel free to contact me if you have any questions and/or suggestions.
